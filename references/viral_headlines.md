# 爆款标题公式库

本文件定义用于生成爆款标题的公式模板，可用于关键词选题时的网络搜索。

## 一、情绪驱动型

引发好奇、情感共鸣，让读者产生代入感。

| 公式 | 示例 |
|------|------|
| "XX天内，XX万人都在XX" | "3天内，10万人都在用的AI工具" |
| "XX岁前一定要知道的XX件事" | "30岁前一定要知道的5个AI副业方向" |
| "为什么XX都在XX？" | "为什么职场人都在学AI？" |
| "XX之后，我发现了XX" | "使用ChatGPT一年后，我发现了副业新大陆" |
| "XX（现象），说出了多少人的心声" | "AI时代被裁员，说出了多少中产的心声" |
| " XX后，我的XX翻倍了" | "用AI工具3个月后，我的效率翻倍了" |

## 使用说明

1. **选择公式**：根据话题性质选择2-3个适合的公式
2. **填充内容**：将热点话题关键词填入公式
3. **生成变体**：每个话题生成3-5个标题变体
4. **验证效果**：确保标题与实际内容匹配，避免过度标题党
5. **搜索应用**：使用生成的标题在 公众号、知乎、机器之心 平台搜索

## 禁忌词（避免使用）

- 过度夸张：震惊、炸裂、疯了、彻底、全面
- 虚假悬念：万万没想到、竟然、居然
- 无意义刷屏：啊啊啊、哈哈哈、？？？
- 情绪煽动：YYDS、天花板、吊打、秒杀
