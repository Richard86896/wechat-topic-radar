# 选题输出模板

## 保存位置

选题文件保存到 vault：

```text
04.选题决策/每日选题/
```

## 文件命名规则

每日公众号选题排名唯一权威文件：

```text
YYYYMMDD-公众号选题排名.md
```

同一天重复执行时，只更新/覆盖该文件，并在文内追加“更新记录”。不要生成“详细评分表 / 汇总评分表 / 爆款潜力版”等平行文件。

特定关键词选题可使用：

```text
YYYYMMDD-选题-{关键词}.md
```

## Frontmatter 模板

```yaml
---
date: YYYY-MM-DD
generated_at: YYYY-MM-DD HH:mm（北京时间）
data_sources_actual:
  - aihot
  - github_trending
  - brave_search_or_ai_announcements
data_sources_failed: []
scoring_system: wechat-topic-radar 七维120分制
canonical: true
tags:
  - 每日选题
  - 公众号选题
  - 也船长AI
---
```

## 输出要求

1. 每个选题按上述模板格式输出。
2. 每日公众号选题排名只保存/更新一个权威文件。
3. 优先推荐 B 级（75分）以上的选题。
4. 确保原文链接、日期、实际数据源准确。
5. 文件保存后必须运行 `verify_scores.py` 验算七维小计和总分。
