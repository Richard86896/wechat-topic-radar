# 选题输出模板

## 保存位置

选题文件保存到 `04.选题决策/每日选题/` 文件夹下。

## 文件命名规则

```
年月日-每日选题.md
年月日-特定选题.md
```

## 选题输出模板

```markdown
# 选题报告：{选题主题}

## 基本信息
- **选题类型**：每日选题 / 特定选题
- **生成日期**：{yyyy年mm月dd日}
- **选题评分等级**：{S/A/B/C/D}级

## 选题 {n}：{选题主题}

### 1. 选题评分（七维框架）
- **综合评分**：{X}/120（{等级}）
  - 相关性：{X}/20
  - 读者价值：{X}/20
  - 判断空间：{X}/15
  - 落地性：{X}/15
  - 时效性：{X}/10
  - 证据素材：{X}/10
  - 差异化：{X}/10
  - **七维小计**：{XX}（= 前7项之和，必须验算）
  - 加分项：{+X}（最多+20，写明命中了哪几项，每项+4）
  - **总分/120**：{XX}（= 七维小计 + 加分项，必须验算）

### 2. 爆款诊断（八维雷达）
- 数据震撼力：{强/中/弱}
- 故事性：{强/中/弱}
- 争议性：{强/中/弱}
- 实操价值：{强/中/弱}
- 情绪共鸣：{强/中/弱}
- 社交货币：{强/中/弱}
- 标题吸引力：{强/中/弱}
- 可读性：{强/中/弱}
- **短板提示**：{哪个维度最弱，标题/角度应如何补}

### 3. 匹配到的关键词
{关键词1}、{关键词2}、{关键词3}

### 4. 写作角度建议
{核心内容方向、角度、切入点分析}

### 5. 标题建议 TOP5
1. {标题1}
2. {标题2}
3. {标题3}
4. {标题4}
5. {标题5}

### 6. 原文信息
- **热点日期**：{yyyy年mm月dd日}（GitHub Trending 数据截至当日抓取时间，星数可能随时间变化）
- **原文链接**：{URL}

---

## 选题 {n+1}：{选题主题}
...（重复上述结构）
```

## 输出要求
1. 每个选题按上述模板格式输出
2. 文件保存后告知用户文件路径
3. 优先推荐B级（75分）以上的选题
4. 确保原文链接和日期准确

## 综合汇总表格式

最终推荐数量不少于5个。报告结尾必须附汇总表，包含七维小计、加分项、总分、等级，用表格呈现。
