# 公众号人设配置

本文件定义目标公众号的读者画像和内容风格。

## 核心定位

**AI领域职场成长与工具应用及利用AI在职场搞副业**

## 目标读者画像

### 主要读者群体
- **年龄**: 25-45岁
- **职业**: 职场人士（不限行业）、希望提升竞争力的打工人、想开辟副业第二曲线的斜杠青年
- **学历**: 大专及以上
- **兴趣**: AI技术趋势、AI工具应用、职场效率提升、副业变现

### 读者需求
1. 了解AI技术趋势，跟上时代发展
2. 学习AI工具应用，提升工作效率
3. 探索AI副业机会，拓展第二收入曲线
4. 获得可操作的职场成长建议

## 内容风格

### 语调
- 专业但不高冷
- 通俗易懂，避免过度技术术语
- 有观点，有态度
- 实用导向

### 标题风格
- 引发好奇
- 具体明确
- 避免标题党
- 数字量化（"5个技巧"、"3个方法"等）

### 文章结构
- 开头：热点引入，制造共鸣
- 中间：深度分析，多角度解读
- 结尾：实操建议，总结展望

## 内容方向优先级

1. **AI工具应用** - 新工具评测、使用技巧、效率提升方法
2. **AI副业探索** - 副业机会分析、变现路径、实战案例
3. **职场成长建议** - AI时代竞争力提升、职业发展策略
4. **AI行业动态** - 新模型发布、技术趋势、政策解读
5. **实操教程** - 提示词工程、工作流优化、场景应用

## 禁止内容

- 纯新闻报道（无观点无分析）
- 过度娱乐化内容
- 与AI和职场完全无关的话题
- 虚假信息
- 过度营销内容
